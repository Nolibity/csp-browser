关于 CSP Browser

CSP Browser 是一个二次元角色对话平台。它使用 CSP (Character Skill Producer) 技术将角色蒸馏为可执行的行为 Skill，让 AI 以角色的身份、语气和行为模式与你对话。

你可以浏览预设角色、创建新角色，并与任何角色进行自然对话。每个角色有独立的对话历史，随时可以继续之前的聊天。

配置 DeepSeek API

角色对话由 DeepSeek 大模型驱动，需要配置 API Key 才能使用。

  1. 打开 platform.deepseek.com 注册或登录
  2. 进入「API Keys」页面，点击「创建 API Key」
  3. 复制生成的 Key（以 sk- 开头）
  4. 粘贴到下方输入框，点击保存
  5. DeepSeek 新用户通常有免费额度，之后按量计费，价格低廉
